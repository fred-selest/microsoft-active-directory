Mise a jour manuelle

Vous pouvez aussi mettre a jour manuellement en executant:

python updater.py

Ou verifier les mises a jour depuis la ligne de commande:

python updater.py --check

Notes

  • Les fichiers de configuration (.env) sont preserves
  • Les dossiers logs/, data/ et venv/ ne sont pas modifies
  • Les dependances Python sont mises a jour automatiquement
  • Un redemarrage du serveur est necessaire apres la mise a jour

Historique des versions

Version 1.8.0 - Securite et gestion avancee

  • Securite: Reinitialisation de mot de passe, deverrouillage en masse, historique des connexions
  • Integration: Corbeille AD, LAPS (mots de passe admin locaux), BitLocker (cles de recuperation)
  • Gestion: Groupes imbriques, comparaison d'utilisateurs, recherche avancee multi-filtres

Version 1.7.0 - Ameliorations majeures

  • Interface: Tri des colonnes, graphiques Chart.js, support PWA
  • Fonctionnalites: Deplacement utilisateurs, export Excel, duplication de groupes
  • Modules: Cache LDAP, service SMTP, webhooks, export PowerShell

Version 1.6.1 - Corrections

  • Correction encodage UTF-8 pour caracteres accentues
  • Amelioration menu mobile
  • Ajout filtre dans la liste des groupes

Version 1.6.0 - Fonctionnalites avancees

  • API REST complete
  • Systeme d'alertes et favoris
  • Support multi-langues (FR/EN)
  • Modeles d'utilisateurs

Version 1.5.0 - Securite renforcee

  • Protection CSRF
  • Validation force mot de passe
  • Rate limiting
  • Headers de securite
