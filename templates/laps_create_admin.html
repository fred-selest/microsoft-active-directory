Script pour créer le compte administrateur local personnalisé

⚠️ Important :

Ce script génère un script PowerShell à déployer via GPO Preferences.

Le compte sera créé sur tous les ordinateurs ciblés par la GPO.

Ex: LocalAdmin, AdminLAPS, etc.
Annuler

Déploiement via GPO

  1. Ouvrez Gestion de stratégie de groupe
  2. Modifiez la GPO LAPS
  3. Naviguez vers : Configuration ordinateur → Préférences → Paramètres du Panneau de configuration → Utilisateurs locaux
  4. Clic droit → Nouveau → Utilisateur local
  5. Utilisez les paramètres générés ci-dessus
  6. Définissez l'action sur Mettre à jour
