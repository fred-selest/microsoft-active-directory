⚠️ Windows LAPS n'est pas configuré

Windows LAPS est natif dans Windows Server 2022 mais n'a pas encore été activé sur ce domaine.

Pour utiliser LAPS, vous devez :

  1. Activer LAPS via Politique de Groupe
  2. Les ordinateurs enverront automatiquement leur mot de passe
Configuration manuelle (avancé)
Via GPO :
  1. Ouvrir Group Policy Management
  2. Créer une GPO : Computer Configuration > Administrative Templates > System > LAPS
  3. Activer Enable LAPS
  4. Configurer la longueur et l'expiration du mot de passe
Via PowerShell :
# Créer et configurer la GPO LAPS
New-GPO -Name "Windows LAPS" -Comment "Enable Windows LAPS"
Set-GPRegistryValue -Name "Windows LAPS" -Key "HKLM\Software\Policies\Microsoft\Windows\LAPS" -ValueName "EnableLAPS" -Type DWord -Value 1
Set-GPRegistryValue -Name "Windows LAPS" -Key "HKLM\Software\Policies\Microsoft\Windows\LAPS" -ValueName "PasswordLength" -Type DWord -Value 14
New-GPLink -Name "Windows LAPS" -Target "DC=SELEST,DC=local"
